⚙ لوحة الإعدادات الشاملة
1. إدارة العملاء والمواد
إضافة عميل جديد أو تعديل الهاتف أو إضافة مادة وسعر خاصين به.
2. إدارة المصاريف والعمال
إضافة أو حذف بنود المصاريف العامة ومشتريات البضاعة الخارجية وأسماء العمال لصرف اليوميات.
3. إعدادات احتساب أجور العمال
إضافة فئة أجور جديدة أو تعديل الاسم والأجرة والعمال التابعين لها.
4. إدارة المستخدمين وكلمات المرور
الإدمن يتحكم بالكامل. المستخدمون 1-3 للإدخال فقط دون إدارة الإعدادات.
